# AOXYCode - Landing Page com Robô 3D Interativo

Uma Landing Page moderna e imersiva desenvolvida para apresentar uma solução tecnológica de otimização de e-mails e combate ao spam. O grande diferencial é a interface interativa com um robô 3D que acompanha o movimento do cursor do usuário.

## Sobre o Projeto
O objetivo desta página é demonstrar como a tecnologia 3D pode ser integrada ao desenvolvimento web para criar experiências de usuário (UX) memoráveis. O robô atua como um "sentinela", simbolizando a proteção e a inteligência artificial por trás do serviço de entrega de e-mails.

## Tecnologias Utilizadas
* **HTML5/CSS3**: Estrutura e estilização moderna com efeitos de blur e gradientes.
* **[Spline](https://spline.design/)**: Para a integração do modelo 3D interativo via `@splinetool/viewer`.
* **[AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)**: Para as animações de entrada dos elementos conforme o carregamento da página.

## Funcionalidades
* **Interatividade 3D**: O robô central reage em tempo real à posição do mouse.
* **Design Responsivo**: Layout adaptado para diferentes resoluções.
* **Animações Fluídas**: Elementos de texto e botões surgem com efeitos de fade e zoom.
* **Filtro Anti-Spam**: Proposta de valor focada em entregar e-mails diretamente para humanos.

## Como visualizar
1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador de preferência.
